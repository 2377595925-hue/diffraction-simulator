========================================================
   光学衍射仿真 — 使用说明
   Optical Diffraction Simulator — User Guide
========================================================

【推荐方式 — Windows / Mac / Linux 通用】
--------------------------------------------------------
双击 "光学衍射仿真_独立版.html" 即可！
无需安装 Python 或任何软件，直接浏览器打开。
(首次加载需联网获取 Three.js，之后可离线使用)


【服务器版 — 备选方式】
--------------------------------------------------------
运行 diffraction_app.py 启动 Flask 服务器端：
  python diffraction_app.py
然后访问 http://127.0.0.1:8080
支持更多功能：实时参数调节、3D 场景交互、物理量显示。


【Windows 用户 — 备选方式】
--------------------------------------------------------
双击 "launch_windows.bat"（需 Python 3）


【系统要求 (独立版)】
--------------------------------------------------------
• 任意操作系统（Windows / Mac / Linux）
• 现代浏览器（Chrome / Safari / Firefox / Edge）
• 首次打开需联网（加载 Three.js 库）
  → 加载后可离线使用

【系统要求 (服务器版)】
--------------------------------------------------------
• Python 3.8+（下载：https://www.python.org/downloads/）
• 依赖包：flask, numpy, scipy, pillow（自动安装）


【文件说明】
--------------------------------------------------------
光学衍射仿真_独立版.html  ⭐ 推荐！双击打开，全平台通用
diffraction_app.py      → Flask 服务器主程序（备选版用）
launcher_gui.py         → 启动器（备选版用，自动安装依赖并启动浏览器）
launch_windows.bat      → Windows 终端启动
static/                 → 前端界面（备选版用，Flask 自动加载）
requirements.txt        → Python 依赖列表
README.txt              → 本说明文件
